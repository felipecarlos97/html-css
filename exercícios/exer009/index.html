Trabalhando com listas

Listas Ordenadas

Você está fazendo uma lista onde tem uma ordem a ser seguida, quando chega no final, percebe que precisa acrescentar uma coisa no inicio da lista. Para nao ter que mudar os numeros manualmente, poderá escrever a tag <ol> para fazer uma lista ordenada e para cada item da lista, coloca a tag <li> para cada item da lista.

  1. Acordar
  2. Conferir se hoje tem muita demanda
  3. Levantar
  4. Escovar os dentes
  5. Tomar café

Também podemos escolher se a contagem da lista inicia com letras ou números, incluindo o parâmetro "TYPE" dentro da tag, e ainda poedmos escolher por qual número começar com o parâmetro "START", também dentro da tag.

  1. Acordar
  2. Conferir se hoje tem muita demanda
  3. Levantar
  4. Escovar os dentes
  5. Tomar café
  1. Acordar
  2. Conferir se hoje tem muita demanda
  3. Levantar
  4. Escovar os dentes
  5. Tomar café

Listas não Ordenadas

As listas não ordenadas, são as listas onde a ordem não intefere, onde não necessita numerar a sequência. Temos como exemplo uma lista de compras de supermercado. Para esse tipo de lista, utilizamos a tag <ul> e a tag <li> dentro dela.

  • Leite
  • Bolo
  • Bacon
  • Pão

Tambem podemos alterar os marcadores com o parâmetro TYPE depois da tag <ul>.

  • Leite
  • Bolo
  • Bacon
  • Pão
  • Leite
  • Bolo
  • Bacon
  • Pão

Misturando listas

Podemos misturar as listas como desejamos. Ordenada dentro de não ordenada, vice e versa ou um mesmo tipo de lista dentro da outra. Segue os exemplos.

Listas de definição

Para encerrarmos as listas, temos a lista de definição, que serve para definir um termo que desejamos. Usando a tag <dl>, e dentro dela usamos as tags <dt> para o termo e <dd> para a definição.
