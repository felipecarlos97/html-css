Parágrafos e quebras de linhas

No parágrafo para se quebrar uma linha, tem que usar uma tag específica. Essa tag é as letras "br" entre os colchetes angulares.Com essa tag, os parágrafos podem ser quebrados, como no exemplo a abaixo.

HINO DO FLAMENGO
Uma vez Flamengo
Sempre Flamengo
Flamengo eu sempre hei de ser
É o maior prazer... ❤ ⚽

E para conseguir escrever os colchetes angulares,
existem códigos específicos.
&+lt; para < e &+gt; para >.
