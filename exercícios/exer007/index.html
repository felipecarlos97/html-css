Principais formatações

Negrito ou destaque

Podemos colocar uma palavra
ou frase em NEGRITO
utilzando a TAG "b". Essa é
uma TAG não semântica, utilizando
simplesmente a forma!

Também podemos colocar uma
palavra em NEGRITO utilizando
a TAG "STRONG" que significa
FORÇA ou escrever a letra com força.
Essa é uma forma semântica de colocar
a letra em negrito!

Itálico ou Ênfase

Agora para colocar uma palavra
em itálico, podemos usar
a TAG "i", sendo a forma não semântica.

Agora, como forma semântica, temos
a TAG "em" para conseguir colocar
as palavras em itálico.

Texto marcado

Também podemos marcar partes de textos
como se fosse com as canetinhas,
utilizando a TAG "mark".

Podemos colocar cor em uma so marcação
selecionando o estilo dentro de mark
ou podemos selecionar em head, que
todos os marks sejam da mesma cor.

Texto grande e texto pequeno

Para deixar um texto grande, pode-se usar a tag "big", porém ela está obsoleta e não é indicada usar, o indicado é formatar. E para deixar um texto pequeno, deve-se usar a tag "small", ela pode ser usada tranquilamente e não está obsoleta.

Texto deletado

O texto deletado serve para passar um risco na palavra ou frase desejada. Para isso, devemos utilizar a tag "del".

Texto inderido

O texto inserido serve para sublinhar a palavra ou frase desejada. Para isso, existem duas maneiras. A tag "u" é a forma não semântica e a tag "ins" é a forma semântica.

Texto sobrescrito

Serve para elevar letras e números como por exemplo: 517*3

Texto subscrito

Serve para abaixar letras e números como por exemplo: h2o
